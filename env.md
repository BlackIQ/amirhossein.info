# Env variables

## Back-End - API

### Location

The dotfile is stored in here:

```shell
/api/.env
```

### Variables

#### Mongodb

Mongodb connection stuff.

```javascript
MONGO_USER=""
MONGO_PASSWORD=""
MONGO_DATABASE=""
MONGO_COLLECTION=""
```

#### Telegram

Telegram bot token.

```javascript
BOT_TOKEN=""
```

#### Application

Application variables.

```javascript
REACT_APP=""
PORT=8000
```

## Front-End - Client

### Location

The dotfile is stored in here:

```shell
/api/.env
```

### Variables

#### NodeJs Application

Link of NodeJs app.

```javascript
REACT_APP_EXPRESS_APP=""
```
