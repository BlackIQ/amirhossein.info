# Env variables

## Back-End - API

### Mongodb

Mongodb connection stuff.

```javascript
MONGO_USER=""
MONGO_PASSWORD=""
MONGO_DATABASE=""
MONGO_COLLECTION=""
```

### Telegram

Telegram bot token.

```javascript
BOT_TOKEN=""
```

### Application

Application variables.

```javascript
REACT_APP=""
PORT=8000
```

## Front-End - Client

### NodeJs Application

Link of NodeJs app.

```javascript
REACT_APP_EXPRESS_APP=""
```
